== Description ==
###Woocommerce Ajax Grid
A simple WordPress plugin that helps you filter your products by category terms with Ajax

just use this shortcode anywhere
<pre>[woo_ajax_grid]</pre>

###Features
* Visual Composer Support

== Installation ==
1. Upload "woo-grid-ajax-filters.zip\" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
